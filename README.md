
# Morpion

Projet validant du 01/02/2024.
le célèbre jeu du morpion (Tic-tac-toe)


## Run Locally

Clone the project

```bash
  git clone https://github.com/annieherieau/thp_morpion_game
```

Go to the project directory

```bash
  cd thp_morpion_game
```

Install dependencies

```bash
  bundle install
```

Start the application

```bash
  ruby app.rb
```
